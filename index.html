Introdução Como Utilizar Linked Lists Queues Binary Heap Graph

Indrodução

Aqui estão reunidas algumas bibliotecas de estruturas de dados em javascript para uso público. O objetivo do desenvolvimento dessas bibliotecas é apenas o meu estudo pessoal das estruturas de dados aqui listadas. Sinta-se à vontade para utilizá-las.

As estruturas de dados implementadas e disponíveis aqui são:

  • Linked Lists
    • Singly Linked
    • Doubly Linked
  • Queues e Stacks
  • Binary Heap
  • Priority Queues
  • Graphs

Cada uma das estruturas de dados é apresentada aqui com seus métodos e forma de utilização, bem como a sua respectiva marcação de complexidade de tempo e de espaço, com marcações como esta: O(logN) ou O(N). As cores são de acordo com o gráfico:

Como Utilizar

Para utilizar qualquer uma das bibliotecas aqui apresentadas, baixe o arquivo correspondente a à estrutura de dados desejada e importe o arquivo no seu código com o seguinte comando: (usando linkedlist.js como exemplo)

import linkedList.js
